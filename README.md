# Productivity Dashboard

A clean, modern productivity dashboard web application to help you stay focused and organized.

## Features

- **Task Management**: Add, complete, and delete tasks with priority levels (low, medium, high)
- **Focus Timer**: Pomodoro-style timer with preset options (25, 15, or 5 minutes)
- **Daily Statistics**: Track completed tasks, pending tasks, productivity score, and focus time
- **Daily Goals**: Set and track progress on daily goals
- **Data Persistence**: Automatically saves your progress to local storage
- **Responsive Design**: Works on desktop and mobile devices

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Start adding tasks and tracking your productivity!

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and responsive design
- `app.js` - Application logic and interactivity

## Usage

### Task Management
- Enter a task in the input field
- Select a priority level (Low, Medium, High)
- Click "Add Task" to add it to your list
- Click the checkbox to mark a task as complete
- Click the Ã— button to delete a task

### Focus Timer
- Click "Start" to begin the timer
- Click "Pause" to pause the timer
- Click "Reset" to reset to 25 minutes
- Use preset buttons to set different durations

### Daily Goals
- Track your progress on completing 5 tasks
- Track your focus time toward 2 hours
- Manually check off custom goals
